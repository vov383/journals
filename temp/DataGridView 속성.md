# AutoGenerateColumns
자동 생성 컬럼
`DataGridView`의 `AutoGenerateColumns` 속성이 `true`로 설정되어 있으면, 바인딩된 데이터 소스의 모든 필드에 대한 컬럼을 자동으로 생성합니다. 
만약 데이터 소스에 빈 필드나 예상치 못한 추가 필드가 포함되어 있다면, 이로 인해 빈 컬럼이 생길 수 있습니다.
